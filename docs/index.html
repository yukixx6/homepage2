HARIBO

HARIBOとは

  • ９０年以上の歴史を持つグミキャンディのパイオニアで世界最大のメーカー。
  • 日本では三菱食品から発売されている。
  • ハリボーという名前の由来は、ハリボーの創設者のハリボーの創業者Hans Riegel（ハンス・リーゲル）が、ドイツのBonn(ボン)に会社を建てたことから、頭文字を取り(Hans Riegel,Bonn)HARIBOとなった。


代表的なグミ
GOLDBAREN
  • 日本では、ゴールドベアという名前で売られている。
  • とても硬く、この硬さが特徴。
  • 味はとてもフルーツ感が強く、作られた甘さがある。

SCHNECKEN
  • 日本ではシュネッケンという名前で売られている。
  • ラクリッツという甘草植物の木の根より抽出されたエキスを使って作られたグミ。
  • グミがタイヤのように巻かれていて、シュネッケンというのはこの形から来ている。
